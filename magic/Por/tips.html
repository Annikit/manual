Resolução de problemas

O MagicTune poderá não funcionar no caso de o monitor ser substituído ou se o controlador da placa gráfica for actualizado enquanto o MagicTune estiver a ser utilizado. Se assim for, reinicie o sistema.


Verifique quando o MagicTune não funciona correctamente.

* A funcionalidade MagicTune é apenas encontrada em computadores (VGA) com o
  SO Window que suporte Plug and Play.

* Para verificar se o computador está disponível para a funcionalidade
  MagicTune, siga os passos abaixo (sistema operativo Windows XP);

Painel de controlo -> Desempenho e manutenção -> Sistema -> Hardware -> Gestor de dispositivos -> Monitores -> Depois de eliminar o monitor Plug and Play, localize 'Monitor Plug and Play' ao procurar novo hardware.


Visite o Web site do MagicTune para obter suporte técnico para o MagicTune, FAQs (perguntas e respostas) e actualizações de software.
