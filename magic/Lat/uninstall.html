Noòemt instalâciju

TMagicTune programmatûru var nonemt, izmantojot tikai "Pievienot vai nonemt programmas" opciju Windows kontroles Panelî.

Izpildiet sekojoðus soïus, lai noòemtu MagicTune.

  1. Izvçlnç dodieties uz [Task Tray] ' [Start] ' [Settings] un atlasiet [Control Panel]. Ja programma darbojas Windows XP vidç, dodieties uz [Control Panel] izvçlnç [Start].


  2. Kontroles panelî noklikðíiniet uz „Pievienot vai noòemt programmas”.


  3. “Peivienot vai noòemt programmas” ekrânâ atrodiet "MagicTune." Noklikðíiniet uz tâs, lai to izceltu


  4. Lai noòemtu programmu, noklikðíiniet uz „Change/Remove” (Mainît/Noòemt) pogas.


  5. Noklikðíiniet uz „Yes” (Jâ), lai sâktu noinstalçðanas procesu.


  6. Gaidiet lîdz parâdâs dialoga logs „Uninstall Complete” (Noinstalçðana pabeigta).

Apmeklçjiet MagicTune Interneta mâjas lapu, lai iegûtu MagicTune tehnisko atbalstu, atbildes uz jautâjumiem un programmatûras atjauninâðanu
